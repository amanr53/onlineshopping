1. Install Sublime Text 2.0.2 x64 Setup.
2. Install XAMPP Server.
3. Database is stored in "onlinestore.sql".
4. Run XAMPP Control Panel
5. Import "onlinestore.sql" in php myadmin.
6. Unzip the folder in "C:\XAMPP\xampp\htdocs"
7. From XAMPP Control Panel start Apache and MySQL.
8. Run using "localhost/onlineshopping/register.php"
